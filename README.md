# covid19-tracker || Website Link: [covid19-tracker](https://indiancovid19tracker.herokuapp.com/)
coronavirus live updates

# Follow below steps for run this project:-

1. install Node.js
2. install Express
3. install MongoDB
4. install Nodemon
5. Run this project using 'nodemon' command
